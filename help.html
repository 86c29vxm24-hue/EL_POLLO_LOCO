Help

Steuerung

  • Links laufen / Move left: Pfeil links / Arrow left, Button links / Left button
  • Rechts laufen / Move right: Pfeil rechts / Arrow right, Button rechts / Right button
  • Springen / Jump: Pfeil hoch / Arrow up, Leertaste / Space
  • Flasche werfen / Throw bottle: Taste D / Key D, Button D / D button

Ziel

Sammle Coins und Flaschen, besiege die Gegner und triff den Endboss mit Flaschen, um das Level zu gewinnen.

Tipps

  • Beim Springen auf Huehner kannst du sie besiegen.
  • Nutze Flaschen sparsam fuer den Endboss.
  • Achte auf deine Lebensanzeige links oben.

Sound & Anzeige

  • Der Sound-Button schaltet alle Sounds global stumm/an.
  • Falls Aenderungen online nicht sichtbar sind: Browser hart neu laden (Cmd+Shift+R).
